Food Delivery App Developed in Flutter.

Tech:
- Flutter
- GetX State management 
- Flutter Hooks
- Firebase Authentication
- Firebase Firestore
- Firebase Storage

Features:
- Login/Sign Up
- Products View
- Products Detail View
- Add to Cart
- Order Placement
- Previous Orders View
- Previous Order detail view
- Profile View
- Logout
